VITAMIN K2 ・ 維生素K2

維生素K2 是什麼？

一次看懂維生素K2（MK-7）：營養角色、食物來源，以及為什麼常與鈣、D3 一起補充。

30 秒看懂 ・ KEY TAKEAWAYS

  • ①一句話：維生素K2 是維生素K 的一種型式（常見為 MK-7），屬脂溶性維生素。
  • ②功能宣稱：維生素K有助於血液正常的凝固功能，並有助於骨骼的健康。
  • ③常見搭配：鈣＋維生素D3＋K2 是常見的骨骼保健組合。
  • ④食物來源：納豆是維生素K2 含量特別高的代表食物。

一、維生素K2 是什麼？

維生素K 是一群脂溶性維生素的總稱，主要分為 K1（葉綠醌，phylloquinone）與 K2（甲萘醌類，menaquinones）。K1 主要來自綠色蔬菜，K2 則多來自發酵食品與動物性食物，並由腸道菌部分合成。

保健食品中常見的 K2 型式為 MK-7，特點是在體內停留時間較長。

二、維生素K 在人體中的角色

維生素K 最廣為人知的角色，是參與血液凝固的正常功能；此外也與骨骼健康相關。

營養素功能宣稱（依台灣公告用語）：維生素K有助於血液正常的凝固功能；有助於骨骼的健康。

三、維生素K2 的食物來源

  • 納豆：維生素K2（MK-7）含量特別豐富的代表食物。
  • 發酵食品與部分起司。
  • 蛋黃、動物肝臟與肉類（含量較低）。

綠色蔬菜（如菠菜、青花菜）則富含維生素K1。

四、為什麼 K2 常與鈣、D3 一起補充？

在骨骼保健配方中，鈣、維生素D3 與維生素K2 常被一起設計：維生素D 幫助鈣吸收，維生素K 則與鈣的正常利用相關。這也是「鈣＋D3＋K2」成為常見組合的原因。愷樂生醫的 GABA 鈣鎂晶凍 即含維生素D3 與 K2。

五、常見型式與選購

  • MK-7：保健食品最常見的 K2 型式。
  • 複方：常與鈣、維生素D3 一起設計。

選購時留意 K2 型式（如 MK-7）、含量單位為微克(µg)，以及是否與 D3、鈣搭配。

六、注意事項

  • 正在服用抗凝血藥物（如 warfarin）者，補充維生素K 前務必諮詢醫師，因維生素K 會影響藥效。
  • 維生素K2、D3 為脂溶性，隨餐食用助吸收，且不建議長期超量。
  • 孕婦、授乳者與特殊疾病者，補充前請諮詢醫師。

研究整理與證據等級

為了讓資訊更透明，我們依證據強度分級整理。星級越高代表證據越充分；必需營養素的營養角色多有充分證據支持，並有台灣公告之功能宣稱可使用。

維生素 K2 常見問題

維生素K2 有什麼營養角色？

+

維生素K有助於血液正常的凝固功能，並有助於骨骼的健康。

維生素K1 和 K2 有什麼不同？

+

K1 主要來自綠色蔬菜，K2 多來自發酵與動物性食物（如納豆），保健食品常用的 K2 型式為 MK-7。

哪些食物含維生素K2？

+

納豆是含量特別高的代表食物，其他如發酵食品、部分起司、蛋黃與肉類也含有。

吃抗凝血藥可以補維生素K 嗎？

+

正在服用抗凝血藥物（如 warfarin）者，補充維生素K 前務必先諮詢醫師，因維生素K 會影響藥效。

MK-7 是什麼？

+

MK-7 是維生素K2 的一種型式，特點是在體內停留時間較長，是保健食品最常見的 K2 型式。

維生素K2 要和 D3 一起補嗎？

+

常一起補充：D3 幫助鈣吸收、K2 與鈣的正常利用相關，加上鈣即『鈣＋D3＋K2』組合。

K2 和鈣有關係嗎？

+

維生素K 有助於鈣的正常利用，因此常與鈣、維生素D3 一起設計。

納豆以外還有哪些 K2 來源？

+

發酵食品、部分起司、蛋黃與肉類也含維生素K2，但納豆含量特別高。

K2 的 MK-7 和 MK-4 怎麼選？

+

兩者皆為 K2 型式，MK-7 停留時間較長、保健食品較常用；可依產品標示選擇。

維生素K2 一天需要多少？

+

請參考公告之每日參考值與產品標示，單位為微克(µg)。

維生素K2 是素食的嗎？

+

視來源與賦形劑而定，部分 MK-7 由發酵生產、適合素食，以產品標示為準。

維生素K2 什麼時候吃？

+

脂溶性，隨餐（含油脂）食用助吸收。

孕婦可以補維生素K2 嗎？

+

建議諮詢醫師後再決定。

維生素K2 有副作用嗎？

+

一般補充量下耐受性良好，但服用抗凝血藥物者務必先諮詢醫師。
