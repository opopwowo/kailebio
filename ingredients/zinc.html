ZINC ・ 鋅

鋅（Zinc）是什麼？

一次看懂鋅這個與免疫、生長和皮膚健康相關的必需微量元素。

30 秒看懂 ・ KEY TAKEAWAYS

  • ①一句話：鋅是人體必需的微量礦物質，參與多種酵素與生理功能。
  • ②功能宣稱：鋅有助於維持正常免疫功能、生長發育與皮膚健康。
  • ③食物來源：牡蠣、紅肉、海鮮、堅果與全穀。
  • ④適合誰認識：素食者、飲食精緻與重視日常保養的人。

一、鋅是什麼？

鋅（Zinc）是人體必需的微量礦物質，雖然需求量不大，卻是數百種酵素的組成或輔因子，廣泛參與身體的基礎生理功能。

鋅無法大量儲存於體內，需要透過每日飲食持續攝取。

二、鋅在人體中的角色

鋅與免疫、生長發育、皮膚健康及多種代謝過程相關。

營養素功能宣稱（依台灣公告用語）：鋅有助於維持正常味覺與食慾；有助於維持生長發育與生殖機能；有助於皮膚健康；為胰島素及多種酵素的成分；有助於維持能量、醣類、蛋白質與核酸的正常代謝；有助於維持正常的免疫功能。

三、鋅的食物來源

  • 海鮮：牡蠣（含量特別高）、蝦、蟹。
  • 紅肉與內臟。
  • 堅果與種子：南瓜籽、腰果。
  • 全穀類與豆類。

植物性食物中的植酸會影響鋅吸收，因此素食者較需留意鋅的攝取。

四、鋅與左旋麩醯胺酸晶凍

愷樂生醫的 左旋麩醯胺酸晶凍 含有鋅，並搭配 左旋麩醯胺酸、PUREWAY-C 維生素C 等成分。

五、常見型式與選購

  • 有機酸鋅（如葡萄糖酸鋅、檸檬酸鋅）：常見且耐受性一般較佳。
  • 胺基酸螯合鋅。
  • 複方：常與維生素C 一起設計。

選購時留意每份元素鋅含量，並注意不宜長期過量。

六、注意事項

  • 長期高劑量補鋅可能影響銅的吸收，不建議自行長期大量補充。
  • 空腹補鋅可能造成腸胃不適，建議隨餐食用。
  • 鋅可能影響部分藥物（如某些抗生素）吸收，請錯開時間並諮詢藥師。

研究整理與證據等級

為了讓資訊更透明，我們依證據強度分級整理。星級越高代表證據越充分；必需營養素的營養角色多有充分證據支持，並有台灣公告之功能宣稱可使用。

鋅 常見問題

鋅有什麼營養角色？

+

鋅有助於維持正常免疫功能、生長發育與皮膚健康，也有助於維持正常味覺食慾，並為多種酵素與胰島素的成分。

哪些食物含鋅？

+

牡蠣含量特別高，其他如紅肉、海鮮、堅果種子、全穀與豆類也是來源。

素食者需要特別注意鋅嗎？

+

是的，植物性食物的植酸會影響鋅吸收，素食者較需留意鋅的攝取。

補鋅要注意什麼？

+

不宜長期高劑量（可能影響銅吸收），建議隨餐食用，並與部分藥物錯開時間。

鋅一天需要多少？

+

請參考國人膳食營養素參考攝取量，並依個人飲食評估。

鋅什麼時候吃？

+

空腹易腸胃不適，建議隨餐食用。

鋅要和鈣、鐵錯開嗎？

+

大量同時攝取可能互相影響吸收，建議錯開時間。

葡萄糖酸鋅和螯合鋅差在哪？

+

是不同的鋅型式，耐受性與吸收略有差異，可依個人狀況與標示選擇。

鋅過量會怎樣？

+

長期高劑量可能影響銅的吸收，不建議自行長期大量補充。

鋅和免疫有什麼關係？

+

依功能宣稱，鋅有助於維持正常的免疫功能（屬維持正常，非增強或治療）。

鋅和皮膚健康有關嗎？

+

依功能宣稱，鋅有助於皮膚健康。

牡蠣以外哪些食物含鋅？

+

紅肉、海鮮、南瓜籽、腰果等堅果種子、全穀與豆類。

鋅是素食的嗎？

+

視鋅鹽來源與賦形劑而定，以產品標示為準；素食者較需留意鋅攝取。

孕婦可以補鋅嗎？

+

需求與劑量請依醫師建議，勿自行大量補充。
