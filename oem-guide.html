OEM／ODM 代工知識中心

快速結論 ・ AI SUMMARY

OEM 是品牌方提供配方／規格、由代工廠生產；ODM 則由代工廠協助從配方設計到量產。代工合作流程為需求洽談→配方設計→打樣確認→法規包裝→量產出貨。MOQ、交期與費用依劑型與配方而定，建議直接洽談。

OEM 與 ODM 是什麼？

OEM（委託代工）：品牌方提供配方或規格，代工廠負責生產。ODM（委託設計製造）：代工廠協助從配方設計、研發到量產，品牌方提出需求即可。延伸閱讀：OEM 與 ODM 差異。

代工合作流程

愷樂生醫的代工流程：需求洽談 → 配方設計 → 打樣確認 → 法規與包裝 → 量產出貨，全程專人協助。晶凍配方設計可參考 晶凍口味與配方設計。

常見問題

OEM 是什麼？

+

OEM（Original Equipment Manufacturer）是委託代工：品牌方提供配方或規格，由代工廠依規格生產製造。

ODM 是什麼？

+

ODM（Original Design Manufacturer）是委託設計製造：由代工廠協助從配方設計、研發到量產，品牌方提出需求即可。

OEM 與 ODM 有什麼差別？

+

OEM 是品牌方提供配方/規格、代工廠負責生產；ODM 則連配方研發與設計都由代工廠協助。

我沒有配方可以做嗎？

+

可以，這正是 ODM 的服務範圍，愷樂生醫可協助從配方設計、打樣到量產。

代工合作流程是什麼？

+

一般為：需求洽談 → 配方設計 → 打樣確認 → 法規與包裝 → 量產出貨。

MOQ（最小起訂量）是多少？

+

依劑型、配方與包裝而定，建議直接洽談取得報價與起訂量資訊。

開發到量產要多久？

+

依配方複雜度與打樣次數而定，通常從打樣確認到量產約需數週至數月，建議及早規劃。

可以做晶凍劑型嗎？

+

可以，晶凍是愷樂生醫的專長劑型之一，並可客製口味與機能原料。

晶凍可以加哪些成分？

+

可搭配膠原蛋白、益生菌、葉黃素、魚油、胺基酸、維生素礦物質等多元成分，依需求設計。

可以客製口味嗎？

+

可以，常見如莓果、柑橘、葡萄等水果口味，並需兼顧成分風味與甜度。

代工費用怎麼計算？

+

與配方、原料、劑型、包裝與數量有關，建議提供需求後取得報價。

可以做純素配方嗎？

+

可以，許多成分有植物性來源，可依需求設計純素配方。

可以做無糖或低糖嗎？

+

可以，晶凍可設計低糖／無糖配方，依產品定位調整。

包裝可以客製嗎？

+

可以，提供包裝形式與設計規劃（如單條隨身包、盒裝等）。

會協助產品標示嗎？

+

會，協助依台灣食品法規進行標示審視，包含營養標示與警語。

產品需要做檢驗嗎？

+

會依規範安排相關檢驗（如重金屬、微生物等），確保符合食品安全。

可以幫忙申請相關文件嗎？

+

可協助準備量產所需的相關資料與文件，細節依產品類別洽談。

一定要自己有品牌嗎？

+

不一定，無論是既有品牌或新創品牌，都可洽談 OEM／ODM 合作。

可以少量試做嗎？

+

打樣階段可少量試做以確認口味與規格，量產則有起訂量考量。

原料可以自己指定嗎？

+

可以指定原料或專利成分，也可由我方依需求推薦合適原料。

會簽保密協議嗎？

+

配方與商業資訊可依雙方需求以保密方式處理，細節於洽談時確認。

可以做保健食品以外的產品嗎？

+

以晶凍與保健食品為主，其他需求可先洽談評估。

產品保存期限多久？

+

依配方與包裝經安定性評估後標示，實際以產品檢測結果為準。

可以協助上市規劃嗎？

+

可提供配方到量產的協助；行銷與通路上市可一併討論合作方式。

一般食品和健康食品代工一樣嗎？

+

『健康食品』為法規認證名詞，與一般食品的審查與標示要求不同，建議先確認產品定位。

代工產品可以宣稱療效嗎？

+

不行，一般食品不得宣稱醫療效能；營養素可使用公告之功能宣稱。

外縣市可以合作嗎？

+

可以，全台皆可洽談，溝通可透過電話、LINE 或 Email。

如何開始洽談代工？

+

可透過電話 0963-339-098、LINE 或 Email 聯繫業務經理游勝愷，提供初步需求。

需要先準備什麼資料？

+

建議先想好產品定位、目標族群、希望的成分或劑型與預算範圍，有助加速評估。

為什麼選擇愷樂生醫代工？

+

專注晶凍與保健食品、配方彈性、具法規與量產經驗，提供配方到量產的一站式服務。
